# Plataforma Web - La Casita de Tom

Este proyecto consiste en un sistema web dinámico desarrollado en Flask para la gestión del catálogo, filtros y control administrativo de usuarios de la librería "La Casita de Tom".

## 📦 Dependencias Necesarias
Para ejecutar este programa, es necesario contar con:
- Python 3.x instalado en el sistema.
- Flask (Microframework de desarrollo web).

## 🚀 Instrucciones de Ejecución
1. Abrir una terminal en la ruta raíz donde se encuentran los archivos `app.py` y el archivo de usuarios `.txt`.
2. Instalar la dependencia de Flask ejecutando el comando:
   pip install Flask
3. Iniciar el servidor local con el siguiente comando:
   python app.py
4. Abrir un navegador web e ingresar a la dirección local generada:
   http://127.0.0.1:5000/
